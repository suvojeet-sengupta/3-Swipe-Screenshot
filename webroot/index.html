How It Works
1
Touch Detection
A lightweight daemon reads raw touch events from your screen's input device via getevent. It tracks each finger's position in real-time with zero battery drain when idle.
2
Gesture Recognition
When 3+ fingers are detected swiping in the configured direction past the sensitivity threshold, a screenshot is triggered. A cooldown timer prevents accidental double-fires.
3
Native System Screenshot
Triggers Android's built-in screenshot — you get the same animation, sound, notification, and share options as pressing Power + Volume Down. Exactly like LineageOS or stock Android.
NATIVE
⚡
Automatic Fallback
If native screenshot isn't available on your ROM, the module automatically falls back to screencap — saves the image, registers it in Gallery, and shows a notification.
